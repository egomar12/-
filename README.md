# 'Клятва пеплом' — Текстовый квест

## Описание проекта
Данный проект представляет собой интерактивную консольную игру, написанную на языке Python. 

Действие разворачивается в мрачном Санкт-Петербурге 1994 года. Главный герой, Александр Яковлев, мстит криминальной группировке «Егерия» за гибель своей семьи. Игроку предстоит принимать сложные моральные решения, которые формируют личность героя и ведут к одной из нескольких концовок.

### Ключевые особенности:
* **Нелинейный сюжет:** Каждый выбор имеет значение и ведет к разным веткам развития истории (3 уникальные концовки).
* **Динамические характеристики:** Решения влияют на параметры персонажа («Человечность», «Осведомленность», «Доверие напарников»).
* **Система сохранений:** Игру можно сохранить в любой момент и продолжить позже (прогресс записывается в файл `savegame.json`).
* **Атмосферный интерфейс:** Плавный вывод текста с имитацией печатной машинки и система очистки экрана для удобства чтения.

## Системные требования
Для запуска игры требуется установленный **Python 3.6** или новее.
Установка дополнительных сторонних библиотек не требуется, проект использует только встроенные модули Python (`json`, `os`, `sys`, `time`).
